Task Hub

A web application that streamlines project organization and enhances productivity. Built with React and Node.js, it utilizes RESTful APIs and Redux for efficient state management. Users can create, read, update, and delete tasks while benefiting from secure data storage with MongoDB and JWT authentication. The responsive design ensures optimal user experience across devices, featuring task prioritization, deadlines, and progress tracking.

Cloud Notes

A web application for efficient note management, built with React and Node.js. It utilizes RESTful API principles for seamless client-server interaction and integrates MongoDB for secure data storage. Users can create, read, update, and delete notes while benefiting from JWT authentication. The app features a responsive design for optimal use on both desktop and mobile devices.

A dynamic web application providing real-time access to the latest news articles using React and the News API. The app features a responsive design, allowing seamless navigation on both desktop and mobile devices, with a loading bar for improved user feedback during data fetching
